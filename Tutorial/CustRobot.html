7. Customize KUKA&ABB Robot¶

Note

Document of this tutorial is CustomizeRobot.3dm , CustomizeRobot.gh | Downlaod in Github

FUROBOT supports several type of robot,but still a lot of robot cannot be included. So we provide customize robot component for KUKA and ABB(UR is all included). The method to customize the robot is same in ABB and KUKA,so we provide a example on ABB(IRB 120):
