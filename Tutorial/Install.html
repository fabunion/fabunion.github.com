1. Download&Installation¶

  1. From Github to download FUROBOT files

  2. Unzip the FUROBOT.zip, Copy the whole FUROBOT folder to Grasshopper Component Library(for example:”C:\Users\YourUsername\AppData\Roaming\Grasshopper\Libraries”)

  1. Make sure in this folder,no exist previous FUROBOT gha&dll(such as FUROBOT.gha and FUROBOT.dll ).If exists, just delete them.

  2. If you need to use ABB realtime function,you may also need to install ABB Robot Communication Runtime 6.0.exe

  3. Restart Rhino,finish.
